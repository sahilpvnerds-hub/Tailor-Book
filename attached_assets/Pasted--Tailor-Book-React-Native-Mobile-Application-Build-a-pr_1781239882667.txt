# Tailor Book - React Native Mobile Application

Build a production-ready React Native mobile application called "Tailor Book" for tailoring businesses. The app should have a modern premium UI inspired by Blinkit, WhatsApp, and Zomato, with smooth animations, clean typography, professional color schemes, responsive layouts, and excellent user experience.

## Technology Stack

* React Native (Latest Version)
* TypeScript
* Redux Toolkit or Zustand for State Management
* React Navigation
* Node.js Backend
* Express.js API
* MongoDB Database
* Firebase Push Notifications
* JWT Authentication
* Voice Recognition Integration
* WhatsApp Sharing Integration
* Email Sending Integration
* Cloud Storage for Documents

---

# User Roles

## 1. Admin

Admin can:

* View all tailor registrations
* Approve or reject tailor accounts
* Manage tailor profiles
* Manage default pricing
* View customer records
* View measurement records
* View invoices
* Send notifications

## 2. Tailor

Tailor can:

* Register account
* Login after admin approval
* Manage customers
* Add measurements
* Generate invoices
* Send invoices through Email and WhatsApp
* View measurement history
* Search customers

# Voice Measurement Entry

Add microphone button beside each measurement field.

Example:

User taps microphone.

User speaks:

"Chest 38"

Automatically fill:

Chest = 38

User speaks:

"Shoulder 17"

Automatically fill:

Shoulder = 17

Voice recognition should support:

* English
* Hindi
* Gujarati

Convert speech to measurement values automatically.
