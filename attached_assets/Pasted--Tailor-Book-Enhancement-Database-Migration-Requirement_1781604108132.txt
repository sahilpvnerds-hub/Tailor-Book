# Tailor Book - Enhancement & Database Migration Requirements

Project: Tailor Book (React Native + Backend + Database)

Please implement the following changes across the application and create all required database migrations, APIs, validations, and UI updates.

## 1. Global Validations

Apply validation throughout the project.

### Mobile Number

* Allow numbers only
* Exactly 10 digits
* No alphabets or special characters

### Email

* Validate proper email format
* Show error for invalid email

### Password

* Minimum 8 characters
* At least 1 uppercase letter
* At least 1 lowercase letter
* At least 1 number
* At least 1 special character

### Required Fields

* Display proper validation messages
* Prevent form submission until validation passes

# 2. Registration Module

## Email OTP Verification

Registration flow:

1. User enters details
2. OTP sent to email
3. User verifies OTP
4. Registration completes only after successful verification

Rules:

* If email is not verified, do not allow registration
* Show message:
  "Please verify your email address before continuing."
